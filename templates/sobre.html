Sobre

Este aplicativo faz parte de uma disciplina Projeto Integrador de graduação da Universidade Virtual de São Paulo do eixo de Computação. Nosso tema foi desenvolver uma plataforma educacional utilizando tecnologias modernas, incluindo Django, Vue.js e Tailwind CSS. Ele foi criado com o objetivo de fornecer um ambiente interativo e informativo para aqueles que desejam aprender conceitos básicos de investimento em ações.

Funcionalidades Principais:

  1. Aprendizado sobre Investimentos em Ações: Utilizamos uma API da bolsa de valores para fornecer informações atualizadas sobre o mercado de ações, incluindo cotações de ações, gráficos de desempenho.
  2. Questionário sobre Finanças: Oferecemos um questionário sobre finanças pessoais, ajudando os usuários a avaliar seu conhecimento e comportamento financeiro. As respostas fornecidas ajudam a entender melhor o perfil do investidor.
  3. Acesso a Dados Financeiros Diversificados: O aplicativo também oferece acesso a dados de outras fontes importantes, como moedas estrangeiras, índices nacionais, taxas de juros (SELIC) e inflação. Isso permite que os usuários tenham uma visão mais abrangente do cenário econômico e financeiro.

Tecnologias Utilizadas:

  • Django: Utilizamos o framework Django para desenvolver o backend robusto e escalável do nosso aplicativo. Ele nos permite gerenciar usuários, autenticação, banco de dados e integração de API de forma eficiente.
  • Vue.js: Empregamos o Vue.js para criar uma experiência de usuário dinâmica e interativa. Com o Vue.js, podemos desenvolver interfaces de usuário responsivas que melhoram a usabilidade do aplicativo.
  • Tailwind CSS: Adotamos o Tailwind CSS para estilizar nossa aplicação de forma eficiente e consistente. Com sua abordagem baseada em classes, podemos criar rapidamente estilos personalizados e garantir uma aparência moderna e elegante para o aplicativo.

Hospedagem e Deploy:

  • GitHub: Todo o código-fonte do nosso aplicativo está hospedado no GitHub, proporcionando transparência, colaboração e controle de versão eficiente.
  • Heroku: Utilizamos a plataforma de hospedagem Heroku para fazer o deploy do nosso aplicativo de forma simples e rápida. O Heroku oferece escalabilidade automática, facilidade de integração e gerenciamento simplificado de aplicativos web.
